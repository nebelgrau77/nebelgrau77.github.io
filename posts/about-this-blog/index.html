About this blog

Last updated: 2021-02-22

The how and why of this simple blog, in my own words :)

I put this blog together to have a way of writing about things I learned, discovered or figured out, that would look neater than a plain readme.md file in a GitHub repo.

I basically wanted a simple, static website, that would require very little setup and could be hosted on GitHub, as I already had an account there.

I found some good articles, like this one, which guided me through the very first steps in setting up a website with Hugo.
I started with the Hyde theme, which I started tweaking a little to my liking, as soon as I learned, how. And this article was really helpful in understanding what goes where in the Hugo folder structure, together with this one.

The nice thing about the themes and Hugo is that once you understand how things work, you can customize your theme picking up elements from others, as they all share the same basic structure. I have very limited knowledge about HTML and CSS, but I managed to add:

  • really neat looking FontAwesome social media icons
  • the ‘read more…’ link
  • categories
  • comments
    and make the page structure work the way I want it.

Last but not least, I made my own glitchy logo, following this tutorial.

If you need a simple, no-nonsense website for a blog or similar, this combination of Hugo and Github Pages is pretty great, and easy to set up. I will probably make some additional tweaks, but for now this is pretty much it.

Enjoy!
